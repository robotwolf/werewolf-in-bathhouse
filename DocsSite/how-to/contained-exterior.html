Room Pattern

Contained Exterior

Use this when a room should feel outside while still behaving like a normal Ginny room with honest connectors, honest markers, and zero topology lies.

What It Is

An authored room with local sky treatment, facade composition, and threshold staging. Presentation lies. Systems truth does not.

What It Is Not

Not a global sky system, not a full outdoor biome, and not an excuse to put the hallway connector where the hallway is not.

Authoring Stack

Use local sky geometry, authored facade meshes, pavement, signage, practical lights, and a bounded post process if needed. Keep the trick local and theatrical.

System Rule

Staging should still be able to answer normal questions about rooms, markers, and connectors. If the room feels special but queries normally, the pattern is working.

Use this pattern when you want theatrical space, not meteorological truth.
