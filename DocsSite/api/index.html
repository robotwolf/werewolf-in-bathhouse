Who Owns Which Mess

Use this page when you need to know which subsystem should change, which one should stay untouched, and which one should stop trying to solve everyone else’s problems.

Ginny

Owns: layout, room selection, required-path logic, hall approaches, adjacency policy, and generator summaries.

Does not own: final shell art, query helpers, or runtime crowd orchestration.

Likely touchpoints: ARoomGenerator, UGinnyLayoutProfile, UGinnyRoomProfile

Mason

Owns: stock and parametric construction, openings, shell generation, and technique-specific embodiment.

Does not own: room selection, mission semantics, or your lovingly hand-built entry facade.

Likely touchpoints: UMasonBuilderComponent, UMasonConstructionProfile, room stock/parametric settings

Staging

Definition: the semantic query and handoff layer that exposes constructed space as usable room truth for tools, gameplay, and runtime orchestration.

Migration note: older commits and redirect-backed assets may still call this layer Stagehand.

Owns: marker lookup, connector lookup, room filtering, query helpers, probes, signage/debug helpers, and the seam runtime systems use instead of geometry archaeology.

Does not own: topology policy, shell construction, or Gideon’s live crowd decisions.

Likely touchpoints: ARoomModuleBase, URoomGameplayMarkerLibrary, URoomSignageComponent, UStagingDebugVisualizerComponent, UStagingSimulationLibrary

Gideon

Owns: runtime orchestration on top of generated space: arrivals, admission, POIs, crowd flow, runtime fear/state, and management logic.

Does not own: room placement or room truth. He consumes published room truth like a civilized runtime layer.

Likely touchpoints: AGideonDirector, AGideonAdmissionBooth, FGideonNPCRuntimeState, queue markers, and entry-threshold tags.

Open Gideon runtime doc

Butch

Owns: dressing and decorative pass behavior.

Does not own: structural truth, gameplay markers, or pretending a chair is a mission socket because it felt dramatic.

Flo

Owns: future configuration-flow orchestration between larger layout regimes.

Current status: still prophecy, not yet payroll.

Contained exterior pattern reminder: outside by presentation, normal room by systems truth.
