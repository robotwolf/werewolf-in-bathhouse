# UI Content Root

This folder holds all UI-related assets and notes for the Spa-Gothic HUD.

Folders
- Widgets: UMG widget blueprints.
- Materials: base UI materials.
- Materials/Instances: material instances.
- Fonts: imported font assets.
- Textures: UI textures.
- Icons: UI icon textures.
- Data: reference data for themes and fonts.
- Curves: animation and timing curves for UI.
- Blueprints: helper Blueprints for UI.

Notes
- UMG assets (.uasset) must be created in the Unreal Editor.
- These docs are here to guide creation and keep naming consistent.
