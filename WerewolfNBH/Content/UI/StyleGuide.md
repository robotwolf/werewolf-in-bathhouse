# Spa-Gothic UI Style Guide

Mood
- Stylish, mature, confident, sensual via atmosphere rather than stereotype.
- Diegetic-light: signage-like panels that could exist in-world.

Visual Language
- Materials: etched glass, tiled placards, brass accents.
- Geometry: rounded rectangles with hard internal corners and notched dividers.
- Lighting: cool neutral glass with warm brass highlights.
- Motion: slow, humid fade-ins; avoid fast UI motion.

Palette (sRGB hex)
- Glass_Base: #D6D8D6
- Glass_Highlight: #EEF0EF
- Glass_Shadow: #A9AEAA
- Brass_Base: #C8A253
- Brass_Glow: #E1C47A
- Text_Primary: #F2F3F1
- Text_Muted: #C8CBC7
- Alert_Warm: #D08A62
- Alert_Cool: #77A7A1
- Danger: #B85C56
- Fog_Mask: #CFCFCF
- Shadow: #262624

Typography
- Sans: Source Sans 3 or Noto Sans
- Serif: Cormorant Garamond or EB Garamond
- Sizes (px): Label 18, Body 22, Title 28, Number 32

Readability in Steam
- High-contrast text with 1-2 px dark outline.
- Thicker icon strokes at small sizes.
- Stable screen positions and minimal clutter.
